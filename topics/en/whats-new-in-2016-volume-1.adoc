= What's New in 2016 Volume 1

== Topic Overview

=== Purpose

This topic presents control proxies of the new and enhanced features for the Infragistics® Test Automation for WPF (Unified Functional Testing) for 2016 Volume 1 release.

The following table summarizes the new features in Infragistics 2016 Volume 1 controls suite that are now supported in Infragistics Test Automation.
