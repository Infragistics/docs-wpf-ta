= xamDataPresenter - Property Reference

== _xamDataPresenter_   Test Object Properties Reference

=== Introduction

This topic provides reference information about the inspectable test object properties of the  _xamDataPresenter_™ controls.

This topic concerns the test object properties of the following controls:

*  _xamDataGrid_  
*  _xamDataCarousel_  
*  _xamDataCards_
