|====

|1. Selecting a text or a series of texts, then pressing the `Delete` key and typing some text.

|1. H __olding down the__ `Shift` key and Clicking on another location within the text. 

2. Holding the `Shift` while using the arrow keys to move away from the current caret location. 

3. Moving the mouse down and dragging it over a range of characters.
| link:{ApiPlatform}test.controls.editors.xamsyntaxeditor~infragisticswpf.test.controls.editors.xamsyntaxeditorproxy~selecttolocation.html[SelectToLocation]

|Typing or Pasting in any series of characters in _xamSyntaxEditor’s_ editor.
| link:{ApiPlatform}test.controls.editors.xamsyntaxeditor~infragisticswpf.test.controls.editors.xamsyntaxeditorproxy~sendstring.html[SendString]

|Clicking on the x _amSyntaxEditor_ control’s view.
| link:{ApiPlatform}test.controls.editors.xamsyntaxeditor~infragisticswpf.test.controls.editors.xamsyntaxeditorproxy~setactiveview.html[SetActiveView]

|Moving the caret position either via the mouse or arrow keys.
| link:{ApiPlatform}test.controls.editors.xamsyntaxeditor~infragisticswpf.test.controls.editors.xamsyntaxeditorproxy~setcaretlocation.html[SetCaretLocation]

|1. Dragging an existing splitter from one position to another. 

2. Dragging an existing splitter off the syntax editor’s view towards the left or top. 

3. Dragging a Mini-splitter to a position creating a new one.
| link:{ApiPlatform}test.controls.editors.xamsyntaxeditor~infragisticswpf.test.controls.editors.xamsyntaxeditorproxy~setsplitter.html[SetSplitter]

|Pressing down the keyboard’s `Insert` key.
| link:{ApiPlatform}test.controls.editors.xamsyntaxeditor~infragisticswpf.test.controls.editors.xamsyntaxeditorproxy~setupdatemode.html[SetUpdateMode]

|Return an integer value of the character location.
