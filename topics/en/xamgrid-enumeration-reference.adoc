= xamGrid - Enumeration Reference

== Topic Overview

=== Purpose

This topic provides reference information about the common enumerations used for testing the  _xamGrid_™ control.

[[_Ref321412912]]
== xamGrid Test Object Enumerations Summary

[[_Ref322452141]]

=== Enumerations reference summary chart

The following table summarizes the purpose of the xamGrid test object enumeration that is used in the most recordable methods to define on which xamGrid item an end-user interaction is performed.

[options="header", cols="a,a,a"]
|====
|Enumeration|Items|Description

|xamGrid selectable item types
|link:{ApiPlatform}test.controls.grids.xamgrid~infragisticswpf.test.controls.grids.xgrdselectableitemtype.html[xGrdColumn]
|Defines a xamGrid column.

|
|link:{ApiPlatform}test.controls.grids.xamgrid~infragisticswpf.test.controls.grids.xgrdselectableitemtype.html[xGrdRow]
|Defines a xamGrid row.

|
|link:{ApiPlatform}test.controls.grids.xamgrid~infragisticswpf.test.controls.grids.xgrdselectableitemtype.html[xGrdCell]
|Defines a xamGrid cell.

|Sort directions
|link:{ApiPlatform}test.shared~infragisticswpf.test.shared.igsortstatus.html[igNotSorted]
|Unsorts a xamGrid column.

|
|link:{ApiPlatform}test.shared~infragisticswpf.test.shared.igsortstatus.html[igAscending]
|Specifies ascending sort order.

|
|link:{ApiPlatform}test.shared~infragisticswpf.test.shared.igsortstatus.html[igDescending]
|Specifies descending sort order.

|Comparison operators
|link:{ApiPlatform}test.shared~infragisticswpf.test.shared.igcomparisonoperatorfull.html[IGComparisonOperatorFull]
|Enumeration that contains various operators used mainly for filtering in the xamGrid control.
