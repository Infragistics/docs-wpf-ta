= xamDiagramToolbox - Properties Reference

== _XamDiagramToolbox_   Properties Reference

=== Introduction

This topic provides reference information about the inspectable test object properties of the  _XamDiagramToolbox_™ control.

=== Table Verification

When performing a standard checkpoint, the returned result contains a table of the current visible menu items, or the root item if the XamDiagramToolbox is closed.

==== Table Verification Definitions

[options="header", cols="a,a,a"]
|====
|Column name|Data type|Description

|Category
|System.String
|String text name of the category the Item belongs to.

|ItemType
|System.String
|String type name of the Item.

|Item
|System.String
|String title name of the Item.

|IndexInCategory
|System.String
|Integer index of the item within the category.

|IndexOverall
|System.String
|Integer inded of the item among all the items.

|====
