== Supported Ultimate UI for WPF control features

=== Supported control features summary

The following table summarizes the newly supported control features of the {ProductName} 2012 Volume 1 Release.

[options="header", cols="a,a,a"]
|====
|Control|Feature|Description

|xamGrid™
|Copy/Paste
|Added new testable actions: 

* CopyToClipboard 

* PasteFromClipboard 

|
|Column moving and resizing
|Added new testable actions: 

* ResizeColumn 

* MoveColumn 

|
|Custom Filter Dialog
|Interaction with the new dialog is supported.

|
|Column types: 

* DateTimeColumn 

* SparklineColumn 

|Interaction with these column types is now supported.

|
|Column hiding, Column chooser dialog
|Added new testable actions: 

* HideColumn 

* ShowColumn (replay only) 

* OpenHiddenColsMenu 

* CloseHiddenColsMenu 

* GetCheckState (applicable to the GridSortableListBox, a sub-control used by the Column Chooser Dialog) 

* SetCheckState (applicable to the GridSortableListBox, a sub-control used by the Column Chooser Dialog) 

|xamDataChart™
|Point Series
|Interaction with charts of this series type is now supported.

|xamPieChart™
|data slices
|The new tesable action _ClickSlice_ was added.
