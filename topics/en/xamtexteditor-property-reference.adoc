== Topic Overview

=== Purpose

This topic provides reference information about the inspectable test object properties of the xamTextEditor™ control.

== xamTextEditor Test Object Properties Reference

=== Properties reference chart

The table below summarizes the control’s test object properties and the tools with which you can inspect them.

[options="header", cols="a,a,a,a,a"]
|====
|Property Name|Returns …|Visible in Object Spy?|Verifiable with Checkpoints?|Accessible in the code?

|_selectedtext_
|the currently selected text. If nothing is selected then returns empty string.
|
|
|

|_selectionlength_
|the length of the currently selected text. If nothing is selected then returns _0_ .
|
|
|

|_selectionstart_
|the starting location of the current text selection. If nothing is selected, it returns the location of the caret.
|
|
|

|_textlength_
|the length of the current text in the editor.
