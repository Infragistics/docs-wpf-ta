== _xam_  _DoughnutChart_  Testable User Actions

=== Testable user action chart

The following table summarizes the control’s user actions and maps them to the recordable methods that emulate them.

[options="header", cols="a,a"]
|====
|User action|Method

|Clicking on a specified slice of the _xamDoughnutChart_ control.
| link:xamdoughnutchart-recordable-methods.html#_Ref352172238[ClickSlice]

|With replay action returns the value for a given slice.
| link:xamdoughnutchart-replay-only-methods.html#_Ref338265452[GetSliceValue]

|With replay action returns the visible slice count for a given ring.
| link:xamdoughnutchart-replay-only-methods.html#_Ref338277363[GetRingSliceCount]

|With replay action returns a delimited string of all the labels of the visible slices for a given ring.
| link:xamdoughnutchart-replay-only-methods.html#_Ref338277374[GetRingSliceLabels]

|====
