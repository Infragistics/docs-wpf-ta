[[_getseriescount]]
== _GetSeriesCount_

=== Description

The  _GetSeriesCount_  method returns the number of series in the xamDataChart control.

=== Details

No additional details.

=== Parameters

No parameters are used with this method.

*Example*

Display a message box with the number of series:

msgbox(WpfWindow("MainWindow").XamDataChart("columnDataChart").GetSeriesCount)

=== Default setting

No default setting.

=== Possible exceptions

No known exceptions.

[[_GetDataPointCount_Method]]
== _GetDataPointCount_

=== Description

The GetDataPointCount method returns the number of data points in a specified series.

=== Details

This method accepts as a parameter either the series name or index.

=== Parameters

The following table explains the method’s parameters together with some sample code.

[options="header", cols="a,a"]
|====
|*Method Parameter*|*Description*

|Series name
|The series name as a parameter. 

*Example*

Displaying a message box with the number of series: 

msgbox(WpfWindow("MainWindow").XamDataChart("columnDataChart").GetDataPointCount (“series1”))

|Series index
|The series index as a parameter. 

*Example*

Displaying a message box with the number of series: 

msgbox(WpfWindow("MainWindow").XamDataChart("columnDataChart").GetDataPointCount (1))

|====

=== Default setting

No default setting.

=== Possible exceptions

No known exceptions.
