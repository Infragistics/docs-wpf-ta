= xamSpreadsheet - Enumerations Reference

== _XamSpreadsheet_   Enumerations Reference

=== Introduction

This topic provides reference material on the common enumerations used for testing the  _XamSpreadsheet_™ control.

=== SshtCellDataType

This enumeration represents the type of cell data that will be returned from the GetCellData action.

The following table lists all the values of the enumeration.

[options="header", cols="a,a"]
|====
|Enumeration|Description

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtcelldatatype.html[sshtCellTypeValue]
|Value 1 – Represents returning of the value of the cell. So if a formula is being used in the cell this will return the calculated value.

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtcelldatatype.html[sshtCellTypeFormula]
|Value 2 – Represents returning the formula of a cell if it has one, if it does not will return an empty string.

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtcelldatatype.html[sshtCellTypeText]
|Value 3 – Represents returning the formatted text of the cell.

|====

=== SshtExitEditType

This enumeration represents the different means to commit or discard the data when the cell is done editing.

The following table lists all the values of the enumeration.

[options="header", cols="a,a"]
|====
|Enumeration|Description

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtexitedittype.html[sshtUpdateSelectedCells]
|Value 1 – Represents committing the data entered to every currently selected cell.

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtexitedittype.html[sshtCreateArrayFormula]
|Value 2 – Represents committing the data as an array formula, which only affects the active cell range, if more than range of cells is selected.

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtexitedittype.html[sshtUpdateActiveCellOnly]
|Value 3 – Represents committing the data to only the currently active cell.

| link:infragisticswpf.test.controls.grids.xamspreadsheet~infragisticswpf.test.controls.grids.sshtexitedittype.html[sshtDiscardChanges]
|Value 4 – Represents discarding the modified cell data and reverting to the previous data.

|====
