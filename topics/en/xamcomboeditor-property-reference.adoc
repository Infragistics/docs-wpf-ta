= xamComboEditor - Property Reference

== _xamComboEditor_   Test Object Properties Reference

=== Introduction

This topic provides reference information about the inspectable test object properties of the xamComboEditor™ control.

The topic covers only the properties that are specific to this xamEditor. For information about the properties that are common to all the xamEditors, refer to to the following topic: link:xameditors-property-reference.html[xamEditors - Property Reference].
