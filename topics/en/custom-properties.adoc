= Custom Properties

=== Introduction

The topics in this group provide information about the custom properties of all supported controls along with those that are control-specific. Custom properties represent proxy objects extended form the controls in order to access sub items of the controls for testing. More details about the types of custom properties are presented in the following list of topics.
